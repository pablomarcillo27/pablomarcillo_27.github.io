Elaboración

Paso 1

Imprimimos y recortamos las imagenes de nuestra elección.

Paso 2

Colocamos mica para plastificar entre las imagenes y las calentamos con una plancha, ubicando una tela para que no se exceda el calor.

Paso 3

Recortamos las figuras y damos bordes, finalmente podemos agregarles una argolla para tener un llavero o lo dejamos como estampa.
